# 📸 Foto&Moda API
Sistema Full Stack para gerenciamento de portfólio de fotografia de moda.
Backend da plataforma **Foto&Moda**, um sistema de portfólio para fotógrafos de moda.

## 🚀 Tecnologias

- Java
- Spring Boot
- MySQL
- JWT Authentication
- Swagger
- Docker
- AWS

## 📦 Funcionalidades

✔ Cadastro de fotos  
✔ Upload de imagens  
✔ CRUD completo  
✔ Formulário de contato  
✔ Login JWT  
✔ Painel administrativo  

## 📚 Documentação

Swagger disponível em:
/swagger-ui/index.html
